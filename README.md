# Binary-Converter

A C++ program that converts a decimal number to its binary and hexadecimal representations.

## What it does
- Prompts the user to enter a decimal number.
- Prints the binary equivalent (MSB first).
- Prints the hexadecimal equivalent (using uppercase letters A-F).

## How to compile and run
```bash
g++ binary_converter.cpp -o binary_converter.exe
.\binary_converter.exe
```

## Example
Input: 15
Output:
```
Binary: 1111
Hexadecimal: F
```
