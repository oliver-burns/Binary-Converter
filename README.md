# Binary Converter

A simple C++ program that converts decimal numbers to binary and hexadecimal representations.

## Features

- Converts positive integers to binary (MSB first) and hexadecimal (uppercase).
- Includes input validation to handle invalid entries gracefully.
- Easy to compile and run on Windows with g++.

## Prerequisites

- C++ compiler (e.g., g++ from MinGW).
- Windows environment.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/oliver-burns/Binary-Converter.git
   cd Binary-Converter
   ```

2. Compile the program:
   ```
   g++ binary_converter.cpp -o binary_converter.exe
   ```

## Usage

Run the executable and enter a positive integer when prompted:

```
.\binary_converter.exe
```

Example:
```
Enter a positive integer to convert: 15
Binary representation: 1111
Hexadecimal representation: F
```
